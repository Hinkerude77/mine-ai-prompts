# Mine AI-prompts

Dette repository samler mine bedste prompts til ChatGPT, Claude og andre AI-værktøjer.

## Formål

Jeg bruger dette til at:
- gemme prompts
- forbedre prompts over tid
- dokumentere hvad der virker
- dele eksempler med andre

## Kategorier

1. Prompts til læring
2. Prompts til tekstforbedring
3. Prompts til idéudvikling
4. Prompts til automatisering
